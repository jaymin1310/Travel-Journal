Travel Journal

Travel Journal is a React-based web application that displays travel destinations and experiences using reusable components and props. The project focuses on building dynamic user interfaces in React by passing and rendering data through props.

Features
Display travel destination cards
Reusable React components
Dynamic rendering using props
Organized and responsive UI layout
Image, location, date, and description display for each destination
Tech Stack
React.js
JavaScript
HTML
CSS
Functionality
Travel data is stored separately and passed to components using props
Each travel entry is rendered dynamically as a reusable card component
The application demonstrates component-based architecture in React
Purpose of the Project

This project was built to practice React fundamentals such as component creation, props handling, dynamic rendering, and reusable UI design.
